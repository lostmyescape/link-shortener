# link-shortener

Link Shortener - это микросервисное приложение для сокращения ссылок с аутентификацией и авторизацией через JWT.

## О проекте
Проект создан на Go, представляет собой систему для сокращения ссылок, построенную на микросервисной архитектуре. Включает REST API для взаимодействия с клиентами и gRPC для внутренней коммуникации сервисов. Для повышения производительности используется кэширование в Redis.

#### Проект состоит из двух микросервисов:

## url-shortener:
- Сокращение URL и их хранение, удаление, редирект
- Безопасный доступ к api запросам
- Проверка подлинности jwt токена
- Настроенное соединение с gRPC, доступ к его методам
- Хранение данных URL в PostgreSQL

## sso:
- Авторизация пользователей
- Генерация jwt/refresh токенов
- Обновление токенов через refresh token механизм
- Кэширование refresh токена в Redis
- Проверка подлинности refresh токена

## Требования
- Go 1.24
- Docker
- Redis
- PostgreSQL

## Запуск проекта
1. Клонируйте репозиторий:
```bash
git clone https://github.com/lostmyescape/link-shortener.git
```
2. Обновите зависимости:
```bash
go mod tidy
```
3. Запуск проекта с Docker Compose:
```bash
docker-compose up --build
```
